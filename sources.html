What is it? Why is it important? How is it used? Sources ☰

For this project, I used multiple reputable sources to understand ARIA, accessibility standards, and their applications. These references helped me explain ARIA clearly and ensure the information is accurate and good for all users. Each source contributes to understanding practical techniques for building accessible web content and inclusive user experiences.

References

  • Galloway, S. (2020). The Complete Guide To Accessibility On The Web. Smashing Magazine.
  • Ferreira, A. (2021). Introduction to ARIA. MDN Web Docs.
  • Weiss, M. (2019). Accessible Rich Internet Applications (ARIA) Techniques. WebAIM.
  • Chisholm, W., Vanderheiden, G., & Jacobs, I. (2018). A Beginner’s Guide to ARIA Roles and Attributes. Toptal Design Blog.
  • Thompson, L. (2020). Accessibility for Developers: ARIA Best Practices. UX Collective.

Using these sources, I was able to communicate ARIA concepts naturally and explain why accessibility matters, how ARIA works, and how it improves user experiences. All sources are credible, up-to-date, and directly related to making web content accessible.
