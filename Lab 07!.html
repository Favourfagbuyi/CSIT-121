Reflection

Creating this candy tier list helped me understand how HTML tables work and how to style them using CSS. I learned how to add structure with headers, rows, and columns while also using color to make the layout more readable and unique. This project made me feel more comfortable linking multiple pages together and creating a consistent design across both pages. it was fun to categorize my favorite candies and see how they stack up against each other visually. Overall, this lab was a great exercise in web development basics and design principles. I have had experience with HTML before, but this lab helped solidify my understanding of tables and styling. 10/10 in my book!
