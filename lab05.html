Lab 05 Reflection

In Lab 05 I created a responsive layout using CSS grid and media queries. On mobile it’s a single column, but on desktop it switches to three columns side by side.

Best design practices include keeping the layout simple, making sure text is easy to read, and designing for both small and large screens. Responsive design matters because people use different devices all the time.
